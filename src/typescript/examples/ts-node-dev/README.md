## Notes

### Options

#### `--respawn`

ファイルの変更を監視するには `--respawn` を使用。

#### `--notify`

`--notify` オプションを使用するには node-notifier が必須。

> - `--notify` - to display desktop-notifications (Notifications are only displayed if `node-notifier` is installed).
>
> Reference: https://github.com/whitecolor/ts-node-dev

node-notifier: https://github.com/mikaelbr/node-notifier
