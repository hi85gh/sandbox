## Notes

### Configuration

以下の設定を追加する。

- `tsconfig.json`
  - `compilerOptions.baseUrl`
  - `compilerOptions.paths`
- `npm run-script` のコマンド
  - `--require=tsconfig-paths/register` または `-r tsconfig-paths/register`
