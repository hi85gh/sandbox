# vue.js/vite

- Vite:
  [Repository](https://github.com/vitejs/vite)

## Setup

```sh
$ npm init vite-app vite

$ cd vite

$ npm install

$ node_modules/.bin/vite -v
vite v1.0.0-rc.4
```
