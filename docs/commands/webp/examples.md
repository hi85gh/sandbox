## 画像を WebP に圧縮

`cwebp` コマンドを使用。

```sh
$ cwebp input_file.png -o output_file.webp
```

Reference: <https://developers.google.com/speed/webp/docs/cwebp>

## WebP を画像に解凍

`dwebp` コマンドを使用。

```sh
$ dwebp input_file.webp -o output_file.png
```

Reference: <https://developers.google.com/speed/webp/docs/dwebp>
