## Notes

<!-- prettier-ignore -->
> WebP（ウェッピー）は、米Googleが開発しているオープンな静止画像フォーマット。ファイルの拡張子は「.webp」。
> Reference: [WebP - Wikipedia](https://ja.wikipedia.org/wiki/WebP)

### Support browsers

Safari 14、iOS Safari 14 リリース以降すべてのモダンブラウザーが対応。

- Can I use...: <https://caniuse.com/#search=WebP>

## Installation

画像の変換には公式で用意されたライブラリを利用。

```sh
# Use Homebrew
$ brew install webp
```
