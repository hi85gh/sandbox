## グローバル IP アドレスの確認

CLI から JSON 形式で IP アドレスの情報を取得。

- IPinfo.io: <https://ipinfo.io/>
  - Example:
    ```sh
    $ curl https://ipinfo.io
    ```
  - Commentary: <http://kumonchu.com/linux/ipinfo/>
- ipify: <https://www.ipify.org/>
  - Example:
    ```sh
    $ curl -s https://api.ipify.org?format=json
    ```
  - Commentary: <https://dev.classmethod.jp/articles/retrieve-json-from-public-api/>
